## 💡 Usage Tips

### **High Contrast**
```dart
// ✅ GOOD: Text on dark
Text('Hello', style: TextStyle(color: textColor))  // White on dark zinc

// ❌ AVOID: Accent on accent
Text('Hello', style: TextStyle(color: accentColor))  // Pink on purple
```

### **Gradient Backgrounds**
```dart
// ✅ GOOD: Use for large CTAs
Container(
  decoration: BoxDecoration(gradient: brandGradient),
  child: Text('Sign Up Now'),
)

// ⚠️ CAREFUL: Too many gradients
// Limit to 1-2 gradient elements per screen
```

### **Glow Effects**
```dart
// ✅ GOOD: Active states
boxShadow: primaryGlow(opacity: 0.5)

// ❌ AVOID: Every element
// Reserve glows for important elements
```

```dart
Container(
  decoration: BoxDecoration(
    gradient: neonGradient,  // or any gradient
  ),

```

### **Use a Glow**
```dart
Container(
  decoration: BoxDecoration(
    boxShadow: accentGlow(opacity: 0.4, blur: 16),
  ),
)
```

### **Mix Colors**
```dart
// Primary with accent
LinearGradient(
  colors: [primaryColor, accentColor],
)

// Triple combo
LinearGradient(
  colors: [accentMint, neonBlue, accentPurple],
)
```

## 📊 Accessibility

All colors meet WCAG AA standards when used properly:

- ✅ **White text** (`textColor`) on all dark surfaces
- ✅ **Zinc-400** (`subtitleColor`) for secondary text
- ✅ **Accent colors** for interactive elements only
- ⚠️ **Never** use accent colors for body text
